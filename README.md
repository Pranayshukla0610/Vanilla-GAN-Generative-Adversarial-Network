# Vanilla-GAN-Generative-Adversarial-Network
Overview

This project implements a Vanilla Generative Adversarial Network (GAN), consisting of a Generator and a Discriminator that compete in a minimax game. The Generator learns to create realistic synthetic data, while the Discriminator learns to distinguish between real and generated samples.

Results

The Vanilla GAN learns the underlying data distribution and generates realistic synthetic samples through adversarial learning between the Generator and Discriminator.
